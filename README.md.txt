# AI Prompt Builder Chrome Extension

A modern AI-powered Chrome extension that helps users:

- Rephrase text
- Summarize content
- Fix grammar
- Improve tone
- Shorten text
- Expand writing
-Basically enhances your prompt

Built using:
- JavaScript
- Chrome Extension Manifest V3
- Shadow DOM
- Modern Glassmorphism UI

## Features

- Beautiful floating AI popup
- Works on any website
- One-click writing enhancement
- Copy & replace support
- Lightweight and fast

## Installation

1. Clone this repository

```bash
git clone https://github.com/nightgalebandit/ai-prompt-builder.git
```

2. Open Chrome Extensions

```txt
chrome://extensions
```

3. Enable Developer Mode

4. Click "Load unpacked"

5. Select the project folder
